# va-spec 1.0.0.connect.2024-04.1

> Released: 2024-05-22

## Release Notes

This is a pre-release version which was presented at GA4GH Connect 2024 in Ascona, CH

Some of the key items included are:
* Draft of the Core Information Model (Core-IM)
* Cohort Allele Frequency Profile
* Pathogenicity Profile
* Therapeutic Response Profile
* Oncogenicity Profile

All content in this version is at a DRAFT maturity level.

## Summary

- **Total schemas:** 9
- **Draft:** 9
