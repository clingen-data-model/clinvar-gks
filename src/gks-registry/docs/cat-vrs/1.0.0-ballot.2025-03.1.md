# cat-vrs 1.0.0-ballot.2025-03.1

> Released: 2025-03-14

## Release Notes

# 1.0.0 Ballot Pre-Release March 2025
This ballot pre-release is for PRC and community review use in moving Cat-VRS to an officially approved 1.0.0 trial use release.

## What's changed
* Documentation improvements (how-tos, diagrams, reorganized outline, ...)
* Expanded examples and test cases
* Adoption of the GKS Maturity Model process and labels
* The following schemas have been promoted from `Draft` to `Trial Use`:
  - CategoricalVariant
  - DefiningAlleleConstraint
  - DefiningLocationConstraint
  - CopyCountConstraint
  - ProteinSequenceConsequence
  - CanonicalAllele
