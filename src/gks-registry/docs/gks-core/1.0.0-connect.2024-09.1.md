# gks-core 1.0.0-connect.2024-09.1

> Released: 2024-09-18

## Release Notes

This is a pre-release version prepared for the GA4GH 12th Plenary Connect (2024) in Melbourne, AUS.

All content in this version is at a DRAFT maturity level.

## Summary

- **Total schemas:** 24
- **Draft:** 24

## Schemas

| Schema | Maturity | Description |
|--------|----------|-------------|
| [Agent](https://github.com/ga4gh/gks-core/blob/1.0.0-connect.2024-09.1/schema/core-im/json/Agent) | draft | An autonomous actor (person, organization, or software agent) that bears some form of responsibility for an activity taking place, for the existence of an entity, or for another agent's activity. |
| [Characteristic](https://github.com/ga4gh/gks-core/blob/1.0.0-connect.2024-09.1/schema/core-im/json/Characteristic) | draft | An object holding a name-value pair used to describe a trait or role of an individual member of a StudyGroup. |
| [Code](https://github.com/ga4gh/gks-core/blob/1.0.0-connect.2024-09.1/schema/data-types/json/Code) | draft | Indicates that the value is taken from a set of controlled strings defined elsewhere. Technically, a code is restricted to a string which has at least one character and no leading or trailing whitespace, and where there is no whitespace other than single spaces in the contents. |
| [Coding](https://github.com/ga4gh/gks-core/blob/1.0.0-connect.2024-09.1/schema/data-types/json/Coding) | draft | A structured representation of a code for a defined concept in a terminology or code system. |
| [CombinationTherapy](https://github.com/ga4gh/gks-core/blob/1.0.0-connect.2024-09.1/schema/domain-entities/json/CombinationTherapy) | draft | A therapeutic procedure that involves multiple different therapeutic procedures performed in combination. |
| [ConceptMapping](https://github.com/ga4gh/gks-core/blob/1.0.0-connect.2024-09.1/schema/data-types/json/ConceptMapping) | draft | A mapping to a concept in a terminology or code system. |
| [Condition](https://github.com/ga4gh/gks-core/blob/1.0.0-connect.2024-09.1/schema/domain-entities/json/Condition) | draft | A disease or other medical disorder. |
| [Contribution](https://github.com/ga4gh/gks-core/blob/1.0.0-connect.2024-09.1/schema/core-im/json/Contribution) | draft | An action taken by an agent in contributing to the creation, modification, assessment, or deprecation of a particular entity (e.g. a Statement, EvidenceLine, DataSet, Publication, etc.) |
| [DataSet](https://github.com/ga4gh/gks-core/blob/1.0.0-connect.2024-09.1/schema/core-im/json/DataSet) | draft | A collection of related data items or records that are organized together in a common format or structure, to enable their computational manipulation as a unit. |
| [Disease](https://github.com/ga4gh/gks-core/blob/1.0.0-connect.2024-09.1/schema/domain-entities/json/Disease) | draft | A particular abnormal condition that negatively affects the structure or function of all or part of an organism and is not immediately due to any external injury. |
| [Document](https://github.com/ga4gh/gks-core/blob/1.0.0-connect.2024-09.1/schema/core-im/json/Document) | draft | A collection of information, usually in a text-based or graphic human-readable form, intended to be read and understood together as a whole. |
| [EvidenceLine](https://github.com/ga4gh/gks-core/blob/1.0.0-connect.2024-09.1/schema/core-im/json/EvidenceLine) | draft | An independent, evidence-based argument that may support or refute the validity of a specific proposition. The strength and direction of this argument is based on an interpretation of one or more pieces of information as evidence for or against the target proposition. |
| [Extension](https://github.com/ga4gh/gks-core/blob/1.0.0-connect.2024-09.1/schema/data-types/json/Extension) | draft | The Extension class provides entities with a means to include additional attributes that are outside of the specified standard but needed by a given content provider or system implementer. These extensions are not expected to be natively understood, but may be used for pre-negotiated exchange of message attributes between systems. |
| [Gene](https://github.com/ga4gh/gks-core/blob/1.0.0-connect.2024-09.1/schema/domain-entities/json/Gene) | draft | A basic physical and functional unit of heredity. |
| [IRI](https://github.com/ga4gh/gks-core/blob/1.0.0-connect.2024-09.1/schema/data-types/json/IRI) | draft | An IRI Reference (either an IRI or a relative-reference), according to [RFC3986 section 4.1](https://datatracker.ietf.org/doc/html/rfc3986#section-4.1>`_ and `RFC3987 section 2.1 <https://datatracker.ietf.org/doc/html/rfc3987#section-2.1>`_. MAY be a JSON Pointer as an IRI fragment, as described by `RFC6901 section 6 <https://datatracker.ietf.org/doc/html/rfc6901#section-6). |
| [Method](https://github.com/ga4gh/gks-core/blob/1.0.0-connect.2024-09.1/schema/core-im/json/Method) | draft | A set of instructions that specify how to achieve some objective. |
| [Phenotype](https://github.com/ga4gh/gks-core/blob/1.0.0-connect.2024-09.1/schema/domain-entities/json/Phenotype) | draft | An observable characteristic or trait of an organism. |
| [RecordMetadata](https://github.com/ga4gh/gks-core/blob/1.0.0-connect.2024-09.1/schema/core-im/json/RecordMetadata) | draft | A reusable structure that encapsulates provenance metadata about a serialized data record or object in a particular dataset (as opposed to provenance about the real world entity this record or object represents). |
| [StudyGroup](https://github.com/ga4gh/gks-core/blob/1.0.0-connect.2024-09.1/schema/core-im/json/StudyGroup) | draft | A collection of individuals or specimens from the same taxonomic class, selected for analysis in a scientific study based on their exhibiting one or more common characteristics  (e.g. species, race, age, gender, disease state, income). May be referred to as a 'cohort' or 'population' in specific research settings. |
| [TherapeuticAction](https://github.com/ga4gh/gks-core/blob/1.0.0-connect.2024-09.1/schema/domain-entities/json/TherapeuticAction) | draft | A therapeutic action taken that is intended to alter or stop a pathologic process. |
| [TherapeuticAgent](https://github.com/ga4gh/gks-core/blob/1.0.0-connect.2024-09.1/schema/domain-entities/json/TherapeuticAgent) | draft | An administered therapeutic agent that is intended to alter or stop a pathologic process. |
| [TherapeuticProcedure](https://github.com/ga4gh/gks-core/blob/1.0.0-connect.2024-09.1/schema/domain-entities/json/TherapeuticProcedure) | draft | An action or administration of therapeutic agents to produce an effect that is intended to alter or stop a pathologic process. |
| [TherapeuticSubstituteGroup](https://github.com/ga4gh/gks-core/blob/1.0.0-connect.2024-09.1/schema/domain-entities/json/TherapeuticSubstituteGroup) | draft | A group of therapeutic procedures that may be treated as substitutes for one another. |
| [TraitSet](https://github.com/ga4gh/gks-core/blob/1.0.0-connect.2024-09.1/schema/domain-entities/json/TraitSet) | draft | A set of phenotype and/or disease concepts that together constitute a condition. |
