# va-spec 1.0.0-connect.2024-09.1

> Released: 2024-09-18

## Release Notes

This is a pre-release version prepared for the GA4GH 12th Plenary Connect (2024) in Melbourne, AUS.

All content in this version is at a DRAFT maturity level.

## Summary

- **Total schemas:** 9
- **Draft:** 9

## Schemas

| Schema | Maturity | Description |
|--------|----------|-------------|

| [AssayVariantEffectFunctionalClassificationStatement](https://github.com/ga4gh/va-spec/blob/1.0.0-connect.2024-09.1/schema/profiles/json/AssayVariantEffectFunctionalClassificationStatement) | draft | A statement that assigns a functional classification to a variant effect from a functional assay. |

| [CohortAlleleFrequencyStudyResult](https://github.com/ga4gh/va-spec/blob/1.0.0-connect.2024-09.1/schema/profiles/json/CohortAlleleFrequencyStudyResult) | draft | A StudyResult that reports measures related to the frequency of an Allele in a cohort |
