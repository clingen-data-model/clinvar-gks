# gks-core 1.0.0.connect.2024-04.1

> Released: 2024-05-22

## Release Notes

This is a pre-release version which was presented at GA4GH Connect 2024 in Ascona, CH

All content in this version is at a DRAFT maturity level.

## Summary

- **Total schemas:** 15
- **Draft:** 13
- **Unknown:** 2

## Schemas

| Schema | Maturity | Description |
|--------|----------|-------------|
| [Code](https://github.com/ga4gh/gks-core/blob/1.0.0.connect.2024-04.1/schema/gks-common/json/Code) | draft | Indicates that the value is taken from a set of controlled strings defined elsewhere. Technically, a code is restricted to a string which has at least one character and no leading or  trailing whitespace, and where there is no whitespace other than single spaces in the contents. |
| [Coding](https://github.com/ga4gh/gks-core/blob/1.0.0.connect.2024-04.1/schema/gks-common/json/Coding) | draft | a concept codified by a terminology system. |
| [CombinationTherapy](https://github.com/ga4gh/gks-core/blob/1.0.0.connect.2024-04.1/schema/gks-common/json/CombinationTherapy) | draft | A therapeutic procedure that involves multiple different therapeutic procedures  performed in combination. |
| [Condition](https://github.com/ga4gh/gks-core/blob/1.0.0.connect.2024-04.1/schema/gks-common/json/Condition) | unknown | A disease or other medical disorder. |
| [Disease](https://github.com/ga4gh/gks-core/blob/1.0.0.connect.2024-04.1/schema/gks-common/json/Disease) | draft | A particular abnormal condition that negatively affects the structure or function  of all or part of an organism and is not immediately due to any external injury. |
| [Extension](https://github.com/ga4gh/gks-core/blob/1.0.0.connect.2024-04.1/schema/gks-common/json/Extension) | draft | The Extension class provides VODs with a means to extend descriptions with other attributes unique to a content provider. These extensions are not expected to be natively understood under VRSATILE, but may be used for pre-negotiated exchange of message attributes when needed. |
| [Gene](https://github.com/ga4gh/gks-core/blob/1.0.0.connect.2024-04.1/schema/gks-common/json/Gene) | draft | A basic physical and functional unit of heredity. |
| [IRI](https://github.com/ga4gh/gks-core/blob/1.0.0.connect.2024-04.1/schema/gks-common/json/IRI) | draft | An IRI Reference (either an IRI or a relative-reference), according to `RFC3986 section 4.1  <https://datatracker.ietf.org/doc/html/rfc3986#section-4.1>` and `RFC3987 section 2.1 <https://datatracker.ietf.org/doc/html/rfc3987#section-2.1>`. MAY be a JSON Pointer as an IRI fragment, as  described by `RFC6901 section 6 <https://datatracker.ietf.org/doc/html/rfc6901#section-6>`. |
| [Mapping](https://github.com/ga4gh/gks-core/blob/1.0.0.connect.2024-04.1/schema/gks-common/json/Mapping) | draft | A mapping to a concept in a terminology system. |
| [Phenotype](https://github.com/ga4gh/gks-core/blob/1.0.0.connect.2024-04.1/schema/gks-common/json/Phenotype) | draft | An observable characteristic or trait of an organism. |
| [TherapeuticAction](https://github.com/ga4gh/gks-core/blob/1.0.0.connect.2024-04.1/schema/gks-common/json/TherapeuticAction) | draft | A therapeutic action taken that is intended to alter or stop a pathologic process. |
| [TherapeuticAgent](https://github.com/ga4gh/gks-core/blob/1.0.0.connect.2024-04.1/schema/gks-common/json/TherapeuticAgent) | draft | An administered therapeutic agent that is intended to alter or stop a pathologic process. |
| [TherapeuticProcedure](https://github.com/ga4gh/gks-core/blob/1.0.0.connect.2024-04.1/schema/gks-common/json/TherapeuticProcedure) | unknown | An action or administration of therapeutic agents to produce an effect  that is intended to alter or stop a pathologic process. |
| [TherapeuticSubstituteGroup](https://github.com/ga4gh/gks-core/blob/1.0.0.connect.2024-04.1/schema/gks-common/json/TherapeuticSubstituteGroup) | draft | A group of therapeutic procedures that may be treated as substitutes for one another. |
| [TraitSet](https://github.com/ga4gh/gks-core/blob/1.0.0.connect.2024-04.1/schema/gks-common/json/TraitSet) | draft | A set of phenotype and/or disease concepts that together constitute a condition. |
