# vrs 1.2.1

> Released: 2022-02-09

## Release Notes

This patch version release shores up inconsistencies in the JSON Schema and definitions, and adds a single source-of-truth document from which the JSON Schema and .rst documentation is derived.

## Summary

- **Total schemas:** 30
- **Unknown:** 30
