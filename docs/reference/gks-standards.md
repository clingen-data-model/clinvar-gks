# GA4GH GKS Standards

!!! note "Under Construction"
    This page is under active development. Content will be added in upcoming updates.

This page will provide a brief overview of the GA4GH Genomic Knowledge Standards (GKS) implemented by this project, including VRS (Variation Representation Specification), Cat-VRS (Categorical VRS), and VA-Spec (Variant Annotation Specification). Links to the official specifications and explanations of how each is applied in the ClinVar-GKS pipeline will be included.
